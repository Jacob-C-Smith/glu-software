 ## Download
 To download glu.app, execute the following command
 ```bash
 $ git clone https://github.com/Jacob-C-Smith/glu.app
 ```
 ## Run
 To build on UNIX like machines, execute the following commands in the same directory
 ```bash
 $ cd glu.app
 $ go run main.go
 ```
